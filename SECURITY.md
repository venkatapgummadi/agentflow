# Security Policy

## Supported Versions

The latest minor release of AgentFlow receives security fixes.

| Version | Supported |
| ------- | --------- |
| 1.3.x   | ✅        |
| 1.2.x   | ❌        |
| < 1.2   | ❌        |

## Reporting a Vulnerability

Please report security vulnerabilities privately to
**venkata.p.gummadi@ieee.org**. Do not open a public GitHub issue
for vulnerabilities.

You will receive an acknowledgment within 72 hours. We aim to
release a fix within 14 days for critical issues.

Thank you for responsibly disclosing.
